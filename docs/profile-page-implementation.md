# 档案页面实现文档

## 概述
档案页面展示当前用户创建的所有生辰八字档案，采用美观的卡片式列表设计，支持查看详情、添加新档案等功能。

## 功能特性

### 1. 页面布局
- **渐变头部**：使用紫色渐变背景，显示页面标题和档案数量
- **添加按钮**：头部右侧的半透明添加按钮，支持快速创建新档案
- **卡片列表**：每个档案以独立卡片形式展示，信息层次清晰

### 2. 档案信息展示
每个档案卡片包含以下信息：
- **档案名称**：用户自定义的档案标题
- **性别标识**：彩色标签显示性别（男/女/未知）
- **生日信息**：公历日期和时间，标注农历/公历
- **八字信息**：四柱八字（年月日时）的干支组合
- **创建时间**：档案创建日期

### 3. 交互功能
- **点击查看**：点击档案卡片跳转到八字详情页面
- **添加档案**：通过头部按钮或空状态按钮跳转到时间查询页面
- **下拉刷新**：支持下拉刷新档案列表
- **上拉加载**：支持分页加载更多档案
- **分享功能**：支持分享档案页面

## 技术实现

### 数据结构
```javascript
data: {
  profileList: [],      // 档案列表
  loading: false,       // 加载状态
  page: 1,             // 当前页码
  hasMore: true        // 是否有更多数据
}
```

### 核心方法

#### 1. 数据加载
- `loadProfileList()`: 加载第一页档案数据
- `refreshProfileList()`: 下拉刷新档案列表
- `loadMoreProfiles()`: 上拉加载更多档案

#### 2. 数据格式化
- `formatProfileList()`: 格式化档案列表数据
- `formatDateTime()`: 格式化创建时间显示

#### 3. 用户交互
- `onProfileTap()`: 点击档案项跳转详情
- `onAddProfile()`: 添加新档案

### 云函数集成
调用 `profileManagement` 云函数的 `getProfiles` 接口：
```javascript
wx.cloud.callFunction({
  name: 'profileManagement',
  data: {
    action: 'getProfiles',
    page: 1,
    limit: 20
  }
})
```

## UI设计特点

### 1. 视觉层次
- 使用卡片阴影和圆角创建层次感
- 渐变背景增强视觉吸引力
- 合理的间距和字体大小搭配

### 2. 色彩方案
- **主色调**：紫色渐变 (#667eea → #764ba2)
- **性别标识**：蓝色(男) / 粉色(女) / 灰色(未知)
- **文字层次**：主要文字 #333 / 次要文字 #666 / 辅助文字 #999

### 3. 响应式设计
- 支持不同屏幕尺寸
- 触摸反馈动效
- 加载状态动画

### 4. 八字展示
八字信息采用四个小卡片的形式展示：
```
[年]  [月]  [日]  [时]
甲子  乙丑  丙寅  丁卯
```
