### 2. 关键改进

#### 2.1 原有实现（已移除）

```javascript
// 旧代码：直接调用云函数，手动处理所有逻辑
const result = await wx.cloud.callFunction({
  name: functionName,
  data: {
    workflowType: 'DRAW_CARD',
    parameters: {
      bazi_name: baziName,
      question: this.data.question || ''
    }
  }
});

// 复杂的错误处理逻辑（200+行）
if (result.result && result.result.success) {
  // 处理成功
} else {
  // 处理失败（各种边界情况）
}
```

#### 2.2 新的实现（已完成）

```javascript
// 新代码：使用 FunctionController，自动处理所有流程
const result = await this.functionController.useFunction('wisdom_insight', {
  bazi_name: baziName,
  question: this.data.question || ''
}, {
  showLoading: false,
  autoPayment: true,
  onQuotaInsufficient: () => {
    log.warn('onAIInterpret', '智慧洞见配额不足');
    return true; // 继续显示支付弹窗
  }
});

// 简化的错误处理（30+行）
if (!result) {
  // 调用失败或被取消
  return;
}
```

| 配额检查 | 手动实现 | 自动处理 | **自动化** |
| 支付流程 | 无 | 自动弹窗 | **新增** |
| 错误处理 | 复杂 | 简化 | **易维护** |
