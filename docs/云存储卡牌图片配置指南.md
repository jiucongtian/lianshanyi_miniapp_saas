# 云存储卡牌图片配置指南

## 概述

为了减小小程序包体积，卡牌图片已从本地存储迁移到微信云存储。本文档说明如何正确配置云存储路径。

## 当前配置状态

已完成的工作：
1. ✅ 在 `miniprogram/config/index.js` 中添加了云存储配置
2. ✅ 修改了 `miniprogram/utils/baziImageMap.js` 以使用云存储路径
3. ✅ 在卡牌页面添加了图片加载失败的处理机制

## 云存储文件结构

所有卡牌图片应上传到云存储的 `cards` 文件夹中，文件命名格式为：
```
01_jiazi.png
02_yichou.png
03_bingyin.png
...
60_guihai.png
```

## 配置步骤

### 步骤 1: 获取云存储 fileID 格式

微信云存储的 fileID 格式通常为：
```
cloud://环境ID.云存储桶ID/路径/文件名
```

要获取正确的格式，请按以下步骤操作：

1. 在微信开发者工具中，打开云开发控制台
2. 进入"存储" -> "cards" 文件夹
3. 随便选择一个已上传的图片文件（例如 `01_jiazi.png`）
4. 复制其 fileID，格式类似：
   ```
   cloud://cloudbase-8g06skyf81a65a87.xxxxxxx/cards/01_jiazi.png
   ```
5. 记录其中的 `xxxxxxx` 部分（云存储桶ID）

### 步骤 2: 更新配置文件

根据获取到的实际 fileID，更新 `miniprogram/config/index.js` 中的云存储配置：

```javascript
// 云环境配置
cloud: {
  envId: 'cloudbase-8g06skyf81a65a87', // 云环境ID（已配置）
  cloudStorageId: 'xxxxxxx', // 替换为实际的云存储桶ID
  cardImagesPath: 'cards', // 云存储中卡牌图片的文件夹路径
},
```

### 步骤 3: 更新 baziImageMap.js

修改 `miniprogram/utils/baziImageMap.js` 中的云路径生成逻辑：

```javascript
// 使用云存储路径
const cloudPath = `cloud://${config.cloud.envId}.${config.cloud.cloudStorageId}/${config.cloud.cardImagesPath}/${fileName}`;
```

## 验证配置

配置完成后，请按以下步骤验证：

1. 重新编译小程序
2. 进入卡牌页面
3. 检查控制台日志中的图片路径是否正确
4. 确认卡牌图片能否正常显示

如果图片无法显示，请查看控制台错误信息，并检查：
- fileID 格式是否正确
- 云存储文件是否都已上传
- 文件名是否与代码中的命名规则一致

## 备用方案

如果云存储图片加载失败，代码会自动记录错误日志。你可以临时保留本地图片作为备用，直到云存储配置完全正确为止。

## 图片上传清单

确保以下60个文件都已上传到云存储的 `cards` 文件夹：

- 01_jiazi.jpeg 至 60_guihai.jpeg

可以使用以下命令检查文件列表（在云开发控制台或通过代码）：

```javascript
// 在小程序中测试获取云存储文件列表
wx.cloud.getTempFileURL({
  fileList: ['cloud://环境ID.存储桶ID/cards/01_jiazi.jpeg'],
  success: res => {
    console.log('云存储文件访问成功:', res)

  fail: err => {

  }
})
```

## 常见问题

### Q1: 图片显示不出来怎么办？

A: 请检查：
1. fileID 格式是否正确
2. 云存储权限设置（建议设置为"所有用户可读"）
3. 文件是否已成功上传到云存储

### Q2: 如何批量上传图片到云存储？

A: 可以使用云开发控制台的批量上传功能，或者编写脚本使用云开发 CLI 工具上传。

### Q3: 如何回退到本地图片？

A: 如果需要临时回退到本地图片，只需修改 `baziImageMap.js` 中的 `imagePath` 生成逻辑，使用 `localPath` 即可。

```javascript
// 临时使用本地路径
imagePath: `/static/new_bazi/${fileName}`,
```

## 性能优化建议

1. 云存储图片会被微信自动CDN加速，首次加载后会有缓存
2. 可以考虑启用图片懒加载（已在预览图片中启用）
3. 如果有更高的性能要求，可以考虑压缩图片尺寸
