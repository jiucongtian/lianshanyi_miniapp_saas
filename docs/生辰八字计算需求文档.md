# 生辰八字计算功能需求文档

## 1. 功能概述

实现一个准确的生辰八字计算功能，将用户输入的公历时间转换为传统的生辰八字。

## 2. 功能详细说明

### 2.1 基本功能
- 支持1949年至2050年间的生辰八字查询
- 精确计算年柱、月柱、日柱、时柱
- 处理节气交接时刻的特殊情况
- 处理子时跨日的情况
- 提供清晰的结果展示界面

### 2.2 计算规则

#### 2.2.1 年柱计算
- 以公历年份计算天干地支
- 计算公式：年干 = (年份 - 4) % 10，年支 = (年份 - 4) % 12

#### 2.2.2 月柱计算
- 以节气为界定月份的依据
- 根据年干推算月干
- 使用节气确定月支
- 在节气交接时刻前后24小时内提供警告提示

#### 2.2.3 日柱计算
- 采用甲子纪日法
- 以1900年1月31日为甲辰日为基准进行推算

#### 2.2.4 时柱计算
- 采用12时辰划分
- 子时（23:00-01:00）特殊处理：
  - 23:00-23:59 归属于前一天的子时
  - 00:00-01:00 归属于当天的子时
- 根据日干推算时干

### 2.3 特殊情况处理

#### 2.3.1 节气交接
- 精确到分钟级别判断节气交接
- 在节气交接点前后24小时提供警告提示
- 明确告知用户可能的变化

#### 2.3.2 子时跨日
- 明确区分子时的前后半段
- 提供跨日提示信息
- 确保日柱计算的准确性

#### 2.3.3 边界情况
- 年份范围（1949-2050）的验证
- 节气交接时刻的精确判断
- 时辰交接时刻的处理

### 2.4 用户界面

#### 2.4.1 时间选择界面
- 提供年月日时分的选择
- 显示当前选择的时间
- 提供清晰的操作按钮

#### 2.4.2 结果显示界面
- 清晰展示四柱（年月日时）
- 分别显示天干地支
- 显示原始公历时间
- 必要时显示警告或提示信息
