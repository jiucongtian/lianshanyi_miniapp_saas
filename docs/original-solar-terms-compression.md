## 压缩方案详解

### 数据格式

```
每年数据 = 120个十六进制字符（24个节气 × 5位）
```

### 编码规则

1. 计算从当年1月1日0时0分到节气时刻的总分钟数
2. 转换为十六进制
3. 左边补0至5位

### 示例

原始数据：
```
1583/01/06T02:57  (小寒)
```

压缩过程：
```
5天 × 1440分钟/天 + 2小时 × 60分钟/小时 + 57分钟
= 7200 + 120 + 57
= 7377分钟
= 0x1CD1
= "01cd1"
```
