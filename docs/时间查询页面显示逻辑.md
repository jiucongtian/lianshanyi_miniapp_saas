# 时间查询页面显示逻辑文档

## 1. 页面结构

### 1.1 页面布局
- 顶部导航栏：显示"时间查询"标题和返回按钮
- 页面标题区：主标题"选择查询时间"和副标题提示文本
- 时间选择卡片：包含时间选择器入口
- 底部操作区：确定按钮

### 1.2 时间选择器组件
- 使用弹出层（t-popup）从底部弹出
- 包含四列选择：年、月、日、时辰
- 顶部操作栏：取消、标题、确定按钮
- 底部圆角设计，符合移动端交互规范

### 2.2 时辰显示格式
- 主要文本：显示时辰名称（如"子时"）
- 描述文本：显示具体时间范围（如"23:00-01:00（次日）"）
- 布局方式：两行显示，时辰名称在上，时间范围在下

### 2.3 样式规范
```css
.picker-item {
  line-height: 80rpx;      // 行高
  text-align: center;      // 居中对齐
  font-size: 32rpx;       // 时辰名称字号
  color: #333;            // 时辰名称颜色
  
  .picker-item-desc {
    font-size: 24rpx;     // 时间范围字号
    color: #999;          // 时间范围颜色
    line-height: 1;       // 行高
    margin-top: -10rpx;   // 与上方文字的间距
  }
}
```

## 3. 交互逻辑

### 3.1 选择器打开
- 点击时间选择区域触发
- 显示当前时间对应的默认值
- 子时特殊处理：23:00-01:00 跨日显示

### 3.2 选择过程
- 支持上下滑动选择
- 实时更新选择值
- 四列可独立选择

### 3.3 确认选择
- 点击确定按钮保存选择结果
- 格式化显示：年月日 时辰名（具体时间范围）
- 示例：2024年3月15日 子时（23:00-01:00（次日））

### 3.4 取消选择
- 点击取消按钮关闭选择器
- 点击遮罩层关闭选择器
- 保持原有选择值不变

## 4. 时间范围限制

### 4.1 年份范围
- 起始年份：1949年
- 结束年份：2050年
- 初始值：当前年份

### 4.2 月份范围
- 1-12月
- 初始值：当前月份

### 4.3 日期范围
- 1-31日
- 初始值：当前日期
- 【待优化】需要根据年月动态调整每月天数

### 4.4 时辰范围
- 12个时辰
- 初始值：当前时间对应的时辰

## 5. 样式定制建议

### 5.1 可调整项

- 行高和间距
- 选择器高度（当前480rpx）
