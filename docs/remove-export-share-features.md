# 删除导出和分享功能描述说明

## 问题描述
代码中存在大量对"导出档案数据"和"分享给好友"功能的描述，但这些功能实际上并不存在，需要删除所有相关描述。

## 修改内容

### 1. 权限管理器 (permissionManager.js)
- 删除了 `EXPORT` 和 `SHARE` 权限定义
- 更新了用户类型权限配置，移除导出和分享权限
- 删除了 `canExportProfile()` 和 `canShareProfile()` 方法

### 2. 云函数 (userManagement/index.js)
- 更新了用户升级时的权限配置
- 更新了临时用户配额修复时的权限配置
- 移除了所有权限数组中的 `export` 和 `share`

### 3. 客户端页面
- **profile/index.js**: 更新了临时用户配额修复时的权限配置
- **mine/index.wxml**: 删除了升级引导中的导出和分享功能描述
- **register/index.wxml**: 删除了注册页面中的导出和分享功能描述

### 4. 文档更新
- **数据库文档 (usersdb.md)**: 更新了权限管理说明和用户类型权限详细说明
- **产品需求文档 (user-classification-product-requirements.md)**: 删除了所有导出和分享功能的描述
- **API文档 (userClassification-api.md)**: 更新了用户类型说明和权限示例
- **临时用户配额修复文档**: 更新了权限配置说明

## 修改后的权限配置

### 临时用户 (guest)
- 权限：`["view", "create_limited"]`
- 功能：查看、创建档案（限制数量）

### 普通用户 (normal)  
- 权限：`["view", "create"]`
- 功能：查看、创建档案

### 高级用户 (premium)
- 权限：`["all"]`
- 功能：全部功能

## 影响范围
- ✅ 权限系统：完全移除导出和分享权限
- ✅ 用户界面：删除所有相关功能描述
- ✅ 文档说明：更新所有相关文档
- ✅ 代码逻辑：移除相关权限检查方法

## 注意事项
- 此修改不影响现有功能，只是删除了不存在的功能描述
- 用户升级引导现在只显示实际存在的功能
- 权限检查逻辑更加简洁，只关注实际功能

## 测试建议
1. 检查临时用户和普通用户的权限是否正确
2. 验证升级引导页面是否只显示实际功能
3. 确认注册页面的功能描述是否准确
4. 测试权限检查是否正常工作
