1. 产品定位

核心身份定位

身心游卡牌小程序是以 “生命智慧卡” 为核心载体，融合以下功能的一站式卡牌服务小程序，兼顾普通用户的自我探索与专业用户的服务需求：

  • 🎮 娱乐玩法：轻松抽卡、每日提示、互动体验
  • 🧭 咨询服务：疗愈师/导师可用于专业解读
  • 📚 学习练习：帮助用户从“玩卡”逐步进入“懂卡”
  • 🛍 疗愈小店：相关课程、产品的延展服务

差异化定位

区别于纯娱乐卡牌应用（只有娱乐玩法，无深厚文化知识）和纯报告咨询平台（门槛高、体验不轻松、有距离感），小程序以 “轻报告、重体验、强互动” 为核心风格。

我们提供：

  • 低门槛的卡牌体验
  • 清晰阶梯式服务（体验 → 学习 → 深度探索）
  • 强互动的抽卡、每日能量、专属解读
  • 配合疗愈师的专业服务流程

让用户实现：从 “玩卡牌” → “懂卡牌” → “用卡牌解决问题” 的完整成长旅程。

2. 功能结构总览

小程序整体功能模块如下：

模块 说明
首页 智慧洞见抽卡、每日愈见（规划中）
卡牌 基于牌库信息计算出的专属四张生命智慧卡牌
牌库 用户档案管理（注册50名额/未注册3名额）
我的 个人资料设置、卡牌查看器、小店入口

新增功能：每日愈见（规划中）

每日愈见将基于当天能量，为用户输出：

  • 今日能量主题
  • 行动建议
  • 情绪提示
  • 注意事项

目标是帮助用户了解当天的能量走向与“如何应对”，提供每日可感知的陪伴感。该功能未来将成为独立模块，可在首页醒目位置进入。

3. 功能模块说明

3.1 首页 - 智慧洞见（抽卡）

首页是用户互动的核心区域，功能包括：

  • 输入问题（如“明天适合跑步吗？”）
  • 点击“抽卡”
  • 系统抽取卡牌
  • AI 根据卡牌象征、国学底层逻辑、卡面信息输出解读
  • 给出温暖、有启发性的建议

注意：抽卡功能需要小程序注册才能使用。

图：首页智慧洞见抽卡界面示例

3.2 卡牌

“卡牌”页面并非静态展示，而是基于用户在牌库中填写的出生年月日时生成的四张专属生命智慧卡牌。

图：专属生命智慧卡牌展示界面

四张专属卡牌分别对应四种关系维度：

专属卡牌 含义
自我关系卡 内在思想、情绪、性格倾向、自我认知
家庭关系卡 家庭互动、亲密关系、能力与天赋（橙成员代表自己）
事业关系卡 合适的行业方向、合作伙伴类型、工作优势
时空关系卡 人与环境、周期变化、状态连贯性

这些卡牌的使用价值体现在：

  • 帮助用户照见自我、滋润家庭、绽放事业
  • 结合疗愈师解读，可提供深度洞察路线

这是整个小程序中最具“生命意义”与“个性化价值”的核心模块。

3.3 牌库

牌库用于存储用户档案，并为卡牌计算提供生命智慧信息基础。

名额规则

  • 已注册用户：可保存 50 个档案
  • 未注册用户：仅可保存 3 个档案

牌库功能

创建档案（如“小李”），可填信息：

  • 姓名
  • 性别
  • 出生日期
  • 出生时辰（可选择“时辰未知”）

支持增 / 删 / 改 / 查操作。

关于“时辰未知”

如果用户不清楚出生时辰：

  • 系统不会自动调整逻辑
  • 会在自我关系卡上标注一个问号(?)
  • 提醒疗愈师：“本卡不准确，请勿基于此卡做解读。”

图：牌库档案管理界面示例

4. 使用流程示例

用户想问：明天适合跑步吗？

操作步骤：

  1. 注册 → 登录
  2. 进入首页 → 智慧洞见
  3. 输入问题
  4. 点击抽卡
  5. 系统抽取例如「06 己巳」卡牌
  6. AI 给出解读：“你像揣着一团融融的活力，非常适合去跑步，与能量撞个满怀。”

5. 常见问题(FAQ)

① 不注册可以抽卡吗？

不能。抽卡（智慧洞见）功能需要先注册才能使用。

② 牌库容量是多少？

注册用户：50 个档案；未注册用户：3 个档案。

③ 不知道出生时辰怎么办？

可选择“时辰未知”。系统会在自我关系卡上标注问号(?)，提醒解读时不要使用这张卡。

④ 四张关系卡是怎么来的？

根据牌库输入的出生信息计算 → 生成专属生命智慧卡 → 反映用户四类关系的底层结构。

6. 注意事项

  • 用户档案涉及个人信息，请妥善管理
  • 卡牌解读为启发式，不作专业医疗、法律依据
  • 若用于专业疗愈师服务，请提前获得用户同意
  • 抽卡与四张关系卡为辅助工具，不作为命理断定
